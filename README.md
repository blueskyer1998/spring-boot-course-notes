# Spring Boot 學習筆記
此專案為跟著古君威老師在hahow開的「Java 工程師必備！Spring Boot 零基礎入門」課程，所做的實作練習，紀錄自己學習 Spring Boot 的基本觀念與相關開發流程。

## 課程目標
- 了解 Spring Boot 的核心概念與運作方式
- 掌握常用開發功能（MVC、JDBC、JPA、AOP 以及測試）
- 實作多個章節範例，從零開始熟悉後端開發流程
- 最終獨立打造一個具備基本功能的電商網站

# 開發環境
- JDK：21
- Spring Boot：4.0.0
- Build Tool：Maven
- IDE：IntelliJ IDEA

## 章節目錄
- ch01_認識 Spring Boot
- ch02_深入了解 Spring 框架 - IoC
- ch03_深入了解 Spring 框架 - AOP
- ch04_Spring MVC
- ch05_Spring JDBC
- ch06_Spring Data JPA
- ch07_Spring Boot 單元測試
- ch08_其他 Spring Boot 相關知識
- ch09_增進開發效率的技巧
- ch10_軟體開發必備的知識
- ch11_實戰演練：打造簡易的電商網站

## 如何執行
1. 進入章節資料夾，例如：
    ```ssh
    cd "ch03_深入了解 Spring 框架 - AOP/demo"
    ```
2. 使用 IDE 或命令列啟動：
    ```ssh
   mvn spring-boot:run
    ```
